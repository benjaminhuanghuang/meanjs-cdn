## Install yo and mean.js generator
    $ npm install -g yo
    $ npm install -g generator-meanjs

## Create project
    $ yo meanjs

## Install dependencies
    bower
    grunt-cli

## Run App
    $ grunt


## Change title
    /meanjs-cdn/modules/core/client/views/header.client.view.html


## Generate new module
    $ yo meanjs:crud-module libraries
